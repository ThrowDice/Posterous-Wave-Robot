Posterous-Robot

What is Posterous-Robot?

Posterous-Robot is first a Google Wave robot. It is built by lookon with Google Wave Python Library and Posterous API. The posterous bot helps you to post blog to posterous.com in Google Wave.

How to use Posterous-Robot

It's easy to use Poterous-Robot in Google Wave. You just need to add posterous-robot@appspot to your contact and start a new wave with the robot. After logining to Posterous, you can post blog to Posterous by editing the title and body of the foot wavelet.

A detailed guide: How to write a blog using Google Wave Robot for Posterous.
